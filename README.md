# GradeLens

AI-Powered Student Performance Analytics Dashboard built with FastAPI, PostgreSQL, Next.js, and React.

## Features
- Student Performance Analytics Dashboard
- CSV Upload and Processing
- Predictive Analytics for At-Risk Students
- Attendance and Academic Trend Analysis
- Interactive Data Visualizations
- FastAPI REST APIs
- PostgreSQL Database Integration
- Cloud Deployment with Vercel and Render

## Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Python
- SQLAlchemy
- PostgreSQL (Neon)

### Data & Analytics
- Pandas
- NumPy
- Scikit-learn

## Deployment

Frontend: Vercel

Backend: Render

Database: Neon PostgreSQL
